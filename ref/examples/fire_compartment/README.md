# 예시: 방화구획 면적 기준

이 예시는 법령 질문이 들어왔을 때 Fractal IRAC가 바로 숫자를 말하지 않고, source 확인과 조건 분해를 먼저 수행하는 구조를 보여줍니다.

질문:

```text
방화구획의 면적기준은 얼마마다 해줘야 하는지
```

핵심 분해:

- `I0`: 질문 정식화
- `I1`: 현행 법령 rule source 확인
- `I2`: 건축물 조건 확인
- `I3`: 완화/예외 확인
- `I4`: 답변 구성과 검토

도구 intent:

- `I1`: 법령 원문 검색
- `I2`: 사용자 조건 질문 생성
- `I3`: 예외 조항 검색
